
# ToDo App

The challenge for Todo app in Flutter.


## Features

- Clean Architecture 
- TDD
- Bloc
- Unit Tests
- Integration Tests
- SOLID
- Service Locator
- Dio


## Running Tests

To run project tests you can use the provided Shell files in the project.

OR

Integration test
```bash
  flutter test integration_test
```
Unit test
```bash
  flutter test
```
